# Проект: Место

### Описание
Проект "Место" - это сайт в котором мы можем создать/редактировать свой профиль как в популярных мессенджерах.

В этом проекте из нового я использовал базовый JavaScript чтобы пользователь мог изменять имя и профессию как ему хочется.
